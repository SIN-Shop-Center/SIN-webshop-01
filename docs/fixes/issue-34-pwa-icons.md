# Fix #34 — Branded PWA-Icons (192/512/maskable) statt dunkler Platzhalter

> **Status:** OPEN · **Priority:** HIGH (P1) · **Repo:** `SIN-Shop-Center/SIN-webshop-01`
> **Issue:** https://github.com/SIN-Shop-Center/SIN-webshop-01/issues/34

## Context

The current `app/icon.png`, `app/apple-icon.png` are likely placeholders (dark/sin-logo or default Next.js). PWA-Icons need:
- **192×192, 512×512** for Android
- **180×180** (apple-touch-icon)
- **maskable 512×512** (Android adaptive icon with safe-zone)
- **favicon 32×32, 16×16**

## Step 1 — generate a real icon

```sh
# Option A: open Canva / Figma, design 512x512, export PNG

# Option B: use AI
# (e.g. prompt: "minimalist green-and-white letter S inside a rounded square,
#   e-commerce, modern, flat, no text, no shading")
# Then run: pnpm dlx @vitejs/plugin-vue or sharp-cli

# Option C: use next.js's @vercel/og to render
cat > /tmp/icon.tsx << 'EOF'
import { ImageResponse } from 'next/og'

export const size = { width: 512, height: 512 }
export const contentType = 'image/png'

export default function Icon() {
  return new ImageResponse(
    (
      <div style={{
        width: '100%', height: '100%',
        display: 'flex', alignItems: 'center', justifyContent: 'center',
        background: '#047857', borderRadius: '20%',
        fontSize: 320, color: 'white', fontWeight: 700,
      }}>S</div>
    ),
    { ...size }
  )
}
EOF
# Place in app/icon.tsx, app/apple-icon.tsx
```

## Step 2 — place the icons

```sh
# 1. Replace app/icon.png with 32x32 favicon
cp /Users/jeremy/designs/favicon-32x32.png app/icon.png

# 2. Replace app/apple-icon.png with 180x180 apple touch icon
cp /Users/jeremy/designs/apple-touch-icon-180x180.png app/apple-icon.png

# 3. Add the PWA manifest
# (check if public/manifest.json already exists)
cat app/manifest.ts 2>/dev/null || echo "No manifest.ts yet"
```

## Step 3 — `app/manifest.ts`

```ts
// app/manifest.ts
import type { MetadataRoute } from 'next'

export default function manifest(): MetadataRoute.Manifest {
  return {
    name: 'ShopSIN',
    short_name: 'ShopSIN',
    description: 'Dein Online-Shop für Mode, Beauty, Haushalt und mehr',
    start_url: '/',
    display: 'standalone',
    background_color: '#ffffff',
    theme_color: '#047857',
    icons: [
      {
        src: '/icon-192x192.png',
        sizes: '192x192',
        type: 'image/png',
        purpose: 'any',
      },
      {
        src: '/icon-512x512.png',
        sizes: '512x512',
        type: 'image/png',
        purpose: 'any',
      },
      {
        src: '/icon-maskable-512x512.png',
        sizes: '512x512',
        type: 'image/png',
        purpose: 'maskable',
      },
    ],
  }
}
```

## Step 4 — the actual PNGs in `public/`

```sh
# Drop your generated icons in:

#   public/apple-touch-icon-180x180.png  (iOS bookmark, generated by app/apple-icon.tsx)
```
